# EvolveScan — Local AI Code Review Engine (Public Preview)

EvolveScan ist ein modulares KI-System zur lokalen statischen Codeanalyse.
Dieses Repository enthält eine öffentliche Vorschau, bestehend aus:

- Architekturübersicht
- Dokumentation
- Beispiel-Outputs
- abstrahierten Preview-Snippets

Der vollständige Quellcode bleibt privat, kann jedoch im Rahmen eines technischen Interviews eingesehen werden.

---

## 🚀 Features

- Mini-AI zur probabilistischen Codebewertung
- Meta-Engine zur Entscheidungsaggregation
- modulare Heuristik-Engines (Pattern, Semantic, History)
- lernfähiges Probability-Memory
- evolutionäre Regelbildung
- vollständig offline, keine Cloud-Abhängigkeiten
