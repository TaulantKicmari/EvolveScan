# Training Pipeline — Mini-AI

Die Mini-AI nutzt ein Teacher-Student-Modell.

---

## Phase 1 — Bootstrapping

External AIs liefern:

- Aktionen
- Wahrscheinlichkeiten
- Erklärungen

Diese Daten werden im ProbabilityMemory gespeichert.

---

## Phase 2 — Hybrid Mode

Die Mini-AI trifft eigene Entscheidungen und vergleicht sie mit Teacher-Modellen.

---

## Phase 3 — Autonomous Mode

Die Mini-AI arbeitet vollständig offline:

- eigene Regeln
- eigene Scores
- eigene Optimierungen

Engines werden optional.
