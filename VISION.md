# Vision — EvolveScan

EvolveScan verfolgt das Ziel, statische Codeanalyse neu zu denken.

Statt rein regelbasiert zu arbeiten, kombiniert EvolveScan:

- heuristische Engines
- probabilistische Mini-AI
- evolutionäre Regelbildung
- Teacher-Model-Training

Das Ergebnis ist ein System, das nicht nur Code bewertet,
sondern sich selbst verbessert.

EvolveScan ist ein Forschungsprojekt, das zeigt,
wie KI-gestützte Codeanalyse in Zukunft aussehen kann.
